前言

本篇記錄了如何使用Hexo 和 GitHub Pages架設個人Blog網站。

參考幾篇網路前輩們的文章(感謝網路大神們的分享)，把學習的重點過程記錄下來，所以不會
有太多細節。

背景知識

  • 可以註冊一個github帳號並了解git基本指令
  • 了解Node.js以及如何安裝Node.js套件與npm指令
  • 了解Node.js的express render engine的運作方式(如果不修改Hexo Theme可以忽略)
