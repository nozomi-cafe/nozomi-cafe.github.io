Hexo & Github 介紹

發表於2021-08-11更新於2021-08-11

本篇記錄了如何使用 Hexo 和 GitHub Pages架設個人Blog網站。

前言

參考幾篇網路前輩們的文章，把重點過程記錄下來，避免未來失憶忘記細節。
開始之前，要有以下背景知識才能比較能理解本文內容：

  • 註冊一個github帳號並了解git基本指令
  • 了解Node.js以及如何安裝Node.js套件與npm指令
  • 了解Node.js的express render engine的運作方式(如果不修改Hexo Theme可以忽略)

美化blog 安裝Hexo主題版型

接下來要介紹如何使用hexo設置網站版型，作業環境必須先安裝好Node.js與git環境。
Hexo安裝可以參考Hexo官方文件，以下指令的執行環境是Ubuntu Linux。

Hexo安裝之後，到Hexo Theme找一個順眼的的主題版型來套用到自己的blog吧。下面指令以stun主題為例。

後記

Please tell me who you are.

如果deploy遇到 “Please tell me who you are.”, 可修改檔案 \myblog.deploy_git.git\config

*** Please tell me who you are.
Run

git config –global user.email “you@example.com“
git config –global user.name “Your Name”

to set your account’s default identity.
Omit –global to set the identity only in this repository

開啟分類(新增categories與tags)和標籤(tags)目錄

copy \myblog\themes\stun_config.yml 至 \myblog\source_data\stun.yml
